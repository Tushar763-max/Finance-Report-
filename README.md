# 💰 FinReport — Monthly Financial Report Generator

> A powerful, fully offline personal finance web app. Track income, expenses, debts, bills, savings goals and more — all in one place. No backend, no subscription, no login required.

## 📱 Install as an App (PWA)

FinReport works as a **Progressive Web App** — install it on your phone like a native app, for free!

**On Android (Chrome):**
1. Open your live site in Chrome
2. Tap the 3-dot menu → **"Add to Home screen"**
3. Tap **"Add"** — done! 🎉

**On iOS (Safari):**
1. Open your live site in Safari
2. Tap the **Share** button → **"Add to Home Screen"**
3. Tap **"Add"** — done! 🎉

## ✨ Features

### 🏠 Overview Dashboard
- 4 metric cards — Total Income, Total Expenses, Net Savings, Health Score
- Income vs Expenses bar chart
- Expense breakdown donut chart
- 6-month trend line chart

### 💵 Income Tracker
- Add income sources with categories (Salary, Freelance, Investment, Business)
- Remove entries instantly
- Real-time totals

### 💸 Expense Tracker
- Log expenses by category
- Custom categories — add your own with custom colors
- Real-time totals and charts

### 🎯 Budget Goals
- Set monthly spending limits per category
- Visual progress bars showing usage
- Instant "On track" or "Over budget" alerts
- Goal vs Actual spending chart

### 💳 Debt Tracker
- Track loans, credit cards, and any debt
- See remaining balance, interest rate, estimated monthly payment
- Payoff progress bar per debt
- Stacked bar chart — paid vs remaining

### 📅 Bill Reminders
- Add bills with due dates
- Colour-coded alerts — Overdue / Due soon / On track
- Automatic reminder notification on app open

### 🤖 AI-Powered Spending Insights
- Personalized tips based on your actual data
- Detects overspending, low savings, debt issues
- Income diversification suggestions
- Refreshes with every click

### 💯 Financial Health Score
- Score out of 100 based on 4 factors:
  - Savings Rate (30 pts)
  - Budget Control (25 pts)
  - Debt Load (25 pts)
  - Income Stability (20 pts)
- Visual score ring with colour coding

### 📅 Month-over-Month Comparison
- Compare this month vs last month
- Side-by-side bar chart for Income, Expenses, Savings

### 💱 Currency Converter
- Convert between USD, INR, EUR, GBP, JPY, AUD
- Built-in approximate exchange rates

### 📷 Receipt Scanner
- Upload a receipt photo
- Auto-detects amount and category
- Add directly as an expense with one click

### 📈 Savings Projections
- Enter annual return rate, monthly contribution, starting balance
- See projections for 1, 3, 5, and 10 years
- 10-year growth chart (projected vs contributions only)

### 📊 Summary & Export
- Full monthly financial summary with insights
- Complete transactions table
- **Export to CSV** — open in Excel or Google Sheets
- **Print Report** — save as PDF

## 🔐 Security & Settings

- **PIN Lock** — set a 4-digit PIN to protect the app
- **Dark Mode** — full dark theme support
- **User Profile** — name, email, profile photo
- **Currency selector** — 7 currencies supported
- **Language selector** — English, Hindi, Spanish, French, German
- **Data Backup** — download all your data as JSON
- **Data Restore** — restore from a previous JSON backup
- **Sign In / Sign Up** — email & password or Google (demo)
- **Guest mode** — use the app without creating an account

## 🛠️ Built With

- **HTML5, CSS3, Vanilla JavaScript** — no frameworks needed
- **[Chart.js](https://www.chartjs.org/)** — all charts and visualizations
- **PWA** — manifest + service worker for offline support
- **Single file** — entire app in one `index.html`

## 🚀 Getting Started

### Option 1 — Use the live site
Visit the GitHub Pages link above — nothing to install.

### Option 2 — Run locally
```bash
git clone https://github.com/yourusername/finreport.git
cd finreport
# Open index.html in your browser
```

### Option 3 — Download & open
Download `index.html` and open it directly in any browser.
